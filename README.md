# Flame effect schematic generator

Open `fast_schematic_generator.html` in a browser. Edit the system in the JSON
box at the bottom, re-render, then Download SVG or print the packet to PDF.

- `npm test` — regression suite (Node only, no deps)
- `npm run validate` — strict XML check of the exported SVG (+ PNG if cairosvg installed)
- `CLAUDE.md` — architecture, invariants, and roadmap for working on this with Claude Code
- `reference/` — earlier draw.io versions of the Mk. II system

Compliance output is a self-review aid against the published Burning Man Flame
Effects Guidelines — not FAST approval. Verify all parts marked VERIFY PN
before purchase.
