    The Thermoid Type 75 Butane-Propane Hose is designed for safe and efficient transfer of liquefied propane gas between bulk storage tanks, cylinders, bobtail trucks, and home storage tanks. Featuring a nitrile rubber inner tube and a weather-resistant outer cover, this UL-listed hose provides a working pressure of 350 psi thanks to its four-spiral yarn reinforcement. The hose is pin-pricked to allow for controlled gas diffusion.

    Features

    • Provides a working pressure of 350 psi
    • More flexible than braid reinforced hose
    • Easier to route assemblies
    • Easier to coil and handle
    • Less susceptible to premature failure than braided hose if subjected to torque

    Specifications

    • Nominal I.D.: 1/4 in. (6.35mm)
    • Nominal O.D.: 0.63 in. (15.88mm)
    • Reinforcement spirals: 4
    • Working Pressure: 350 psi (2.41 Mpa)
    • Min. Bend Radius: 1.5 in. (38.10mm)
    • Weight: 0.14 lb/ft (0.22 Kg/m)
    • Temperature range: -40° F to 180° F (-40° C to 82° C)
    • Branding: Thermoid Type 75 LPG Hose UL Gas Hose Issue Number (Quarter) MH12585 350 PSI Maximum WP Made in USA
    • Cover Color: Black
    • Manufactured in: Bellefontaine, OH
    • Construction:
      • Tube: Nitrile, ARPM - Class A
      • Cover: Type 75 - Chlorinated Polyethylene,
        ARPM - Class B: 3/4 in. and 1 in.
        Type 75B - Nitrile Blend: 1/4 in. - 1/2 in.
        Abrasion and weather resistant
    • Certifications: UL 21 FILE MH12585 OR UL 21 FILE MH 12585
